# Portfolio

A personal portfolio website built with React and Vite, featuring a clean and minimalist design.

## Sections

- **Home** — Navigation bar and Hero section with introduction and profile image
- **About** — Personal bio, thesis paper link, source code link, and skill cards 
- **Education** — Academic background 
- **Projects** — Featured projects with tech stacks and links
- **Contact** — Get in Touch with location and social links

## Getting Started

### 1. Install dependencies
```bash
npm install
npm run dev
```

### 2. Start & build
```bash
npm run build
npm run preview
```

## Tech Stack

- React 19
- Vite 7
- Lucide React
- Font Awesome
- CSS
